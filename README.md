# 📦 ChestLogger
![Java](https://img.shields.io/badge/Java-21-orange)
![Maven](https://img.shields.io/badge/build-Maven-blue)

Plugin de log de baús para servidores Minecraft (Paper), com integração ao Discord via ConnectionDC.

---

## 🚀 Sobre

O **ChestLogger** registra interações com baús (itens adicionados/removidos) e envia logs em formato de embed para o Discord.

---

## 📦 Funcionalidades

- 📥 Log de itens adicionados/removidos
- 📍 Registro de localização (X, Y, Z)
- 📦 Suporte a:
  - Baús
  - Barris
  - Ender Chest
  - Shulker Box
- 🎨 Envio de embeds para Discord

---

## ⚙️ Requisitos

- Java 21+
- PaperMC API 1.21+
- Plugin **ConnectionDC**
- Maven para build do plugin
---

## 🔗 Dependência

Este plugin depende do ConnectionDC:

- https://github.com/Deczin/ConnectionDC

No `plugin.yml`:

```yml
depend:
  - ConnectionDC
```
